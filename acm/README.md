# ACM Final Exercise
Advanced Cyber Maneuvers (ACM) is one of Cedarville University's zero-credit 
classes devoted to teaching freshmen the foundations of cybersecurity to prepare
them to excel in their classes.

At the end of the semester, ACM has a final exercise to test the students' skills.

This web application was created for that final exercise. It is intentionally 
vulnerable, of course, so don't deploy it in a production environment. 

If you would like to set this web application up for yourself, set up the Apache
server and put the contents of the acm/website directory in /var/www/html. Then
use database_setup.sql to set up the associated MySQL database.
